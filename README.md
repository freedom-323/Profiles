# How to use?
1. Download the `App_Signing_Certificates.zip` file to your device
2. Unzip the file
3. Import the certifates you need into tools such as **[AltStore](https://github.com/altstoreio/AltStore)** or **FlekStore**
4. Sign the ipa file
